# Change a Japanese name to something like a Greek God

ギリシア神話風の名前を付けるやつです。

## ご使用になる前に

```
$ npm clone https://github.com/ginpei/name-like-greek-gods.git
$ cd name-like-greek-gods
$ npm install
```

## 使い方

こんな感じの書式です。

```
npm start [name [url of Wikipedia]]
```
